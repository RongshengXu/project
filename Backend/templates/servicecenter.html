FAQ for Customers
  1. How to make an order?

    2. Please first set the location, if you have not done that(you only need to do this once unless you want to change to another location).The Main page will list all the restaurant nearby, where you can sort them according to distance or valuation score. Choose a restaurant and click it to go the order page. You can add the food you want to your cart by click the "add to cart" button. You can then check your cart by click on the mychart button in the sidebar on the right.

  2. How to set my location?

    3. Use the nav-bar to go the location page. The location will be set using google map. You only to do this for the first time to use this website, unless you want to change to another location.

  3. How to pay for my order?

    4. You need to have a paypal account to pay for the order online. Click the order for a certain restaurant if you have one. After you adjust the quantity and input the necessary information for the order. you can click the confirm button to go the payment page.

FAQ for Restaurants
  1. How to open a restaurant?

    2. You need to go to the restaurant mode by click the "log as owner of a restaurant" button on the sidebar in the right.Then you can open a new restaurant by click the "Open a restaurant" button.

  2. How to add dishes to a restaurant?

    3. Login as restaurant and then click the restaurant where you want to add dishes. The restaurant page will show you a "add more dishes" button to add more dishes.

  3. How to edit my restaurant?

    4. You can add a restaurant with the same name to edit the information of the restaurant. The edition of the dishes will be same.
